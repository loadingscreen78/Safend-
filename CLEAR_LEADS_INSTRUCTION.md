# Clear Mock Leads Data

## To Start Fresh

If you want to clear any existing mock data and start with an empty leads table:

### Option 1: Browser Console
1. Open browser DevTools (F12)
2. Go to Console tab
3. Type: `localStorage.removeItem('leads')`
4. Press Enter
5. Refresh the page

### Option 2: Application Tab
1. Open browser DevTools (F12)
2. Go to Application tab
3. Click on "Local Storage" in left sidebar
4. Find your site URL
5. Find the "leads" key
6. Right-click and delete
7. Refresh the page

## Now the Table Will:
- ✅ Show empty initially
- ✅ Display only leads YOU create through the form
- ✅ No mock data will appear
- ✅ All your created leads will persist

## Create Your First Lead:
1. Click "New Lead" button
2. Fill out the form
3. Submit
4. Lead appears in table immediately!
